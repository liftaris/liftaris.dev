### Content

Kaio Barbosa-Chifan has been a Software Development Engineer II at Moderna since October 2023, working on the Commercial Digital Innovation & Engagement (CDIE) team. His work spans Moderna's public-facing marketing websites, internal compliance tools, infrastructure automation, and developer experience tooling. Across his tenure, Kaio has authored over 80 merged pull requests across 13 repositories, touching every layer of the stack from Terraform infrastructure-as-code to React and Angular frontends, NestJS backends, and CI/CD pipelines. His primary areas of ownership include the Marketing UI platform (Next.js sites powering Moderna's disease education, product, and HCP engagement portals), the Fair Market Value (FMV) compliance application (Angular + NestJS + SQL Server), the Vaccine Finder feature, and supporting infrastructure services. Kaio's day-to-day tech stack includes TypeScript, React, Next.js, Angular, NestJS, Terraform, AWS (CloudFront, Lambda, S3, ECS, WAF, API Gateway), Docker, Helm, Typesense, Contentful CMS, MongoDB Atlas, and GitHub Actions.
