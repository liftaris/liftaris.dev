### Content

Kaio contributed to Moderna's React component library, a shared design system used across the company's marketing websites. He built the Switch component from scratch, implementing it to match Figma design specifications with full design token support for themed styling across selected/unselected states, hover, click, and disabled variants. He introduced a new "radius-round" corner radius token to support the component's pill shape. He also made significant improvements to the Carousel component — fixing gradient behavior and placeholder UI for hidden slides, adding support for single-slide transitions, fixing the hideContent prop, and building a Carousel Preview component for content editors. Additionally, he fixed a CollapseItem component bug where empty items could still be expanded, and corrected a Modal component's Storybook documentation where props were incorrectly labeled. These six pull requests to the UI kit ensured consistent, accessible component behavior across Moderna's multi-market web properties.
